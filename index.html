Welcome to CrackingTheCode

CrackingTheCode is a website dedicated to teaching its students the basics of programming and technology. It's primary focus is web development. So you can find here tutorials (both text-based and video-based) on a variety of topics like CSS, JavaScript, React, Express and MongoDB that make up the popular MERN stack. It also covers underlying technologies (like APIs for example) that power web applications in general.
